네트워트 기초 과정

IoT기반 스마트 헬스케어 웹서비스 개발 과정

교육 기관 : 그린아카데미 (울산점)

교육 기간 : 2022.12 ~ 2023.07

교육 설명 : IoT기반 스마트 헬스케어 웹서비스 개발 과정을 통해 (html5, css, javascript)를 이용한 웹사이트 구현 설계, JavaScript와 Jquery를 사용하여 사용자 입력 처리, 웹 페이지의 동적인 콘텐츠
업데이트, 애니메이션 효과 등을 구현을 공부하고 제작해 보았습니다.
스프링 프레임워크에서 MVC 모델을 이용하여 웹 페이지를 개발하고 서비스, 컨트롤러, 뷰를 분리해 코드의 재사용성과 유지보수성을 높였습니다.
팀프로젝트에서는 팀장을 담당하였고, 프로젝트의 전체적인 구조와 DB설계 및 완성된 부분에 대한 피드백과 개선방안을 제시하였습니다.
첫번째 프로젝트에서는 SNS로그인을 사용자 인증과 권한 부여를 안전하게 처리하기 위해 OAuth2 프로토콜을 적용하였고, 사용자가 구글 계정으로 웹 서비스에 로그인할 수 있는 기능을 구현했습니다.
두번째 프로젝트에서는 실시간 생체데이터 수집이라는 주제로 라즈베리파이 BMP085(대기압 센서)와 Flask를 통해 생체 데이터를 수집해 차트로 처리하고 MySQL Connect를 이용해 DB에 값을 저장하는 기능을 구현해봤습니다.
마지막 프로젝트는 병원솔루션 SCM 관리 프로세스 구현-공급사 관점 의료용품 수주 프로세스 구현으로 공급사 관점에서 물품을 추가하고 재고 상태 및 판매 현황을 조회할 수 있도록 제작 하였습니다.
